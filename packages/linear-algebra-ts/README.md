## Getting started
### Setup
```sh
git clone https://github.com/Palmaswell/math.git
cd math
yarn
```

### Typescript development
```sh
yarn dev:type
```
### Reasons development
yarn dev:reason
```
### Run test suite for Typescript and Reason in watch mode
```sh
yarn test
```
